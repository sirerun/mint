# petstore

MCP server generated by [mint](https://github.com/sirerun/mint) from an OpenAPI specification.

## Usage

### Build

```bash
go mod tidy
go build -o petstore .
```

### Run with stdio transport

```bash
./petstore --transport stdio
```

### Run with SSE transport

```bash
./petstore --transport sse --port 8080
```

### Configure in Claude Desktop

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "petstore": {
      "command": "/path/to/petstore",
      "args": ["--transport", "stdio"]
    }
  }
}
```

## Tools

- **list_pets** - List all pets
- **create_pet** - Create a pet
- **show_pet_by_id** - Info for a specific pet

## Authentication

Set the `MINT_API_KEY` environment variable with your API key.
